# SideProject-1-market-parser
My first side project in quant finance written in C++. A simple program using CLI and taking in a CSV as input. The program is analysing x stock with different functionalities such as: Highest value, lowest value, VWAP, calculating 1std above VWAP, 1std below VWAP, daily stats and user-specified daily moving average. Additional functionality could be added to calculate other metrics, but decided to stop here for the time being and focus on my next project which is the trading engine on-top of the market-parser (stay tuned!). Added a CSV file in case anyone would like to try it out without the hassle of creating your own CSV, enjoy!
