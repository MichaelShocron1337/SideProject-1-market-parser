# SideProject-1-market-parser
My first side project in quant finance. A simple program using CLI and taking in a CSV as input. The program is analysing x stock with different functionalities such as: Highest value, lowest value, VWAP, calculating 1std above VWAP, 1std below VWAP, daily stats and user-specified daily moving average. Additional functionality could be added to calculate other metrics, but decided to stop here because the learning-gain became redundant at this point since the most vital parts are in place, i.e any additional funcitonality wouldn't yield any new challenges. Added a CSV file in case anyone would like to try it out without the hassle of creating your own CSV, enjoy!
